**Digital Contacts Manager** is a Python application designed to manage contacts efficiently. It allows you to add, update, delete, view, and search contacts, along with additional utilities for sending feedback and accessing official contact information.

## ‚ú® Key Features

| Feature | Description |
|---------|------------|
| **Add Contact** | Add a new contact with Name, Age, Address, Phone, and a unique ID. |
| **Delete Contact** | Delete any contact using its unique ID to prevent duplication or errors. |
| **View Contacts** | Display all contacts in a numbered and formatted list with complete details. |
| **Search Contacts** | Search for contacts by name, displaying results in a well-organized table including the ID. |
| **Update Info** | Update any contact either partially or fully re-enter all fields. |
| **Send Feedback** | Submit feedback or suggestions directly to a separate Feedback file. |
| **Contact Us** | View all official communication channels including email, phone, and social media. |
